    推論模型

    1. 新增推論項目

    1. 從左方工作列進入 「 推論項目 」 。
    2. 點擊 「 新增推論項目 」 。
    3. 為推論項目命名。
    4. 選擇訓練好的模型。
    5. 選擇推論資料集。
    6. 點擊 「 開始推論 」 。

    {% hint style="info" %} 上傳推論資料請參考此頁面。 {% endhint %}
