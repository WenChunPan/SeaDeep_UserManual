# 分類資料集

## 查看資料集

1. 從左方工作列進入 「 <mark style="color:blue;">資料集</mark> 」 。
2. 確認所在位置在**資料集內**的 「 訓練項目 」 後，點擊放大鏡圖示的 「 <mark style="color:blue;">檢視細節</mark> 」 。

{% hint style="warning" %}
需查看**推論資料集**，步驟2改為進入 「 推論項目 」 即可。
{% endhint %}
