# 檢視推論結果

## 檢視細節

1. 從左方工作列進入 「 <mark style="color:blue;">推論項目</mark> 」 。&#x20;
2. 點擊放大鏡圖示的 「 <mark style="color:blue;">檢視細節</mark> 」 。

<figure><img src="../../../.gitbook/assets/image (89).png" alt=""><figcaption><p>檢視細節</p></figcaption></figure>

## 2. 確認推論結果

點擊類別按鈕，即可切換並查看預測為該類別的圖片。

{% hint style="success" %}
點擊圖片即可放大查看
{% endhint %}
